


The equal-width method divides the range of the data into equal-width bins.

Data range:
Max value = 91, 
Min value = 10, 
Range = 91 - 10 = 81


Number of bins = 3

Width of a bin:
\[
\text{Width} = \frac{\text{Range}}{\text{Number of bins}} = \frac{81}{3} = 27
\]


The bins are:

- Bin 1: [10, 37)
- Bin 2: [37, 64)
- Bin 3: [64, 91]

The values in each bin are:

1. Bin 1: {10, 16, 25, 38} 
2. Bin 2: {42, 44, 45, 58, 61}
3. Bin 3: {72, 77, 84, 91}
